# v0.3.5 - if-assign in function bodies (2026-04-28)

## What changed

Added support for `if <compare>: target = expr` (conditional assignment
with no else branch) as a non-final statement in function bodies compiled
by `compileFuncBodyExpr`. The assignment target must be a local that was
already declared before the `if` statement.

## Pattern

```python
def f(a, b):
    x = a
    if a < b:
        x = b
    return x
```

## Bytecode

Same condition block as `isIfReturn` (COMPARE_OP(cond)+POP_JUMP_IF_FALSE+
NOT_TAKEN). The then-body emits LOAD_FAST (for Name RHS) or walkExpr,
then STORE_FAST. POP_JUMP_IF_FALSE jumps over the then-body.
