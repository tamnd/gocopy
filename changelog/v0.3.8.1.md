# v0.3.8.1 — upgrade to gopapy v0.4.1 (2026-04-28)

Bumps the parser dependency from `github.com/tamnd/gopapy v0.2.11` to
`v0.4.1`. This is a patch release: no new language features, no fixture
changes, no observable behavior change.

## What changed in gopapy v0.4.1

The parser was fully rewritten internally. From gocopy's perspective the
public API is compatible:

- All AST type names are unchanged (Module, FunctionDef, If, Return, Name,
  Constant, BinOp, Compare, UnaryOp, Assign, AugAssign, …).
- All field names are unchanged (P.Line, P.Col, Body, Orelse, Args, …).
- `Module` gained a `TypeIgnores []TypeIgnore` field (always empty);
  gocopy ignores it.
- A new `version.VersionFeature` bitmask was added to the module but is not
  used by gocopy.

## Fixtures

198/198 fixtures byte-identical — no regressions.
