# v0.1.16 - function bodies with general expressions (2026-04-27)

Extends the compiler to handle multi-statement function bodies where
expressions are recursively composed of parameter and local-variable
Name nodes, BinOp, and UnaryOp (USub/Invert).

## Supported patterns (new)

```python
def f(a, b):
    x = a + b
    return x

def f(a, b):
    return a + b

def f(a):
    return -a

def f(a, b, c):
    return a + b + c

def f(a, b, c):
    x = a + b
    y = x + c
    return y
```

## Architecture

New classification kind `modFuncBodyExpr` routes single function
definitions with a general body to `compileFuncBodyExpr`.

`extractFuncBodyExpr` in classify_ast.go validates the function:
- Only positional params, no defaults / *args / **kwargs
- Body: zero or more `local = expr` assignments followed by one `return expr`
- Each statement on a distinct source line (no semicolons)
- All Name nodes refer to params or previously assigned locals
- Total locals (params + body vars) ≤ 15 (LFLBLFLB nibble constraint)

`compiler/func_body.go`: `compileFuncBodyExpr` + `funcState.walkExpr`:
- LOAD_FAST_BORROW_LOAD_FAST_BORROW (opcode 87) when both BinOp children
  are direct Name references: oparg = (left_slot << 4) | right_slot
- LOAD_FAST_BORROW (opcode 86) for single-name loads and the right child
  of nested BinOps
- STORE_FAST (opcode 112) for local assignments
- BINARY_OP + 5 cache words for arithmetic
- UNARY_NEGATIVE / UNARY_INVERT for unary operations

`bytecode/opcode.go`: adds `LOAD_FAST_BORROW_LOAD_FAST_BORROW Opcode = 87`.

Linetable: RESUME at def line uses SHORT0(1)[0,0); first instruction
of each body statement uses appendFirstLineEntry (ONE_LINE1 for the
common delta-1 case); subsequent instructions on the same statement line
use appendSameLine. The module-level wrapper reuses FuncDefModuleLineTable.

LocalsPlusKinds: 0x26 per parameter (CO_FAST_LOCAL|CO_FAST_ARG),
0x20 per body-assigned local (CO_FAST_LOCAL).

- `bytecode/opcode.go`: `LOAD_FAST_BORROW_LOAD_FAST_BORROW = 87`

- 6 new fixtures: 144-149

## Out of scope for this version

- Constants (int, float, True, False, None) in function body expressions
- Same-line bodies (semicolon-separated) which use STORE_FAST_LOAD_FAST
- Functions with no parameters (no LOAD_FAST_BORROW path needed)
- Default arguments, *args, **kwargs
- More than 15 total locals
